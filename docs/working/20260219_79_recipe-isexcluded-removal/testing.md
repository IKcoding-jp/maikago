# テスト計画

## テスト戦略

### 手動テスト（主要）
レシピ取り込みは Cloud Functions + GPT API を経由するため、手動での動作確認が中心。

#### テストケース1: 調味料を含むレシピの取り込み
1. レシピ入力画面で「肉じゃが」のレシピテキストを入力
2. 材料に醤油、みりん、砂糖、料理酒が含まれるテキストを使用
3. **期待結果**: 確認画面で調味料を含む全材料が表示される
4. 「追加する」ボタンを押す
5. **期待結果**: 調味料を含む全材料が買い物リストに追加される

#### テストケース2: 基本的な食材のみのレシピ
1. シンプルな食材のみのレシピ（サラダ等）を入力
2. **期待結果**: 全食材が確認画面に表示され、追加される

#### テストケース3: 確認画面での手動削除
1. レシピを取り込む
2. 確認画面で不要な材料（水など）をゴミ箱アイコンで削除
3. 「追加する」ボタンを押す
4. **期待結果**: 削除した材料以外が買い物リストに追加される
