# 要件定義

**Issue**: #79
**作成日**: 2026-02-19
**ラベル**: bug

## ユーザーストーリー
ユーザー「レシピを取り込んだのに、醤油やみりんなどの調味料が買い物リストに入っていない」
アプリ「レシピから抽出した全材料をリスト化し、ユーザーが確認画面で不要なものだけ削除できるようにする」

## 要件一覧
### 必須要件
- [ ] Cloud Functions の parseRecipe プロンプトから `isExcluded` 除外ルール（ルール5）を削除する
- [ ] プロンプトの出力JSON定義から `isExcluded` フィールドを除去する
- [ ] `RecipeConfirmScreen._onAdd()` の `if (ingredient.isExcluded) continue;` を削除する
- [ ] レシピ取り込み時に調味料を含む全材料が確認画面に表示され、追加されること

### オプション要件
- [ ] `RecipeIngredient` モデルから `isExcluded` フィールドを完全削除（後方互換性を考慮し、残してもよい）

## 受け入れ基準
- [ ] 「肉じゃが」のレシピで醤油・みりん・砂糖・料理酒が確認画面に表示される
- [ ] 確認画面で「追加する」を押すと調味料を含む全材料が買い物リストに追加される
- [ ] 既存のレシピ取り込みフロー（テキスト入力→解析→確認→追加）が正常に動作する
- [ ] `flutter analyze` でエラーがないこと
