# 要件定義

**Issue**: #81
**作成日**: 2026-02-19
**ラベル**: enhancement

## ユーザーストーリー
ユーザー「タブの追加とリストの追加、どっちのボタンがどっちかわからない…」
アプリ「両方のボタンにテキストラベルを表示して、一目で区別がつくようにします」

## 要件一覧
### 必須要件
- [ ] タブ追加ボタン（AppBar右上）にテキストラベル「タブ追加」を表示する
- [ ] リスト追加FAB（画面右下）を `FloatingActionButton.extended` に変更し「リスト追加」ラベルを表示する
- [ ] 両ボタンのアイコンも変更し、視覚的な区別を強化する

### オプション要件
- [ ] 画面幅が狭い場合のレスポンシブ対応（ラベルの省略・アイコンのみ表示等）

## 受け入れ基準
- [ ] タブ追加とリスト追加のボタンが、テキストラベルにより一目で区別できる
- [ ] 各テーマ（pink, blue等）で視認性が損なわれない
- [ ] 既存の操作フロー（タップ→ダイアログ表示）に変更がない
- [ ] Web/モバイル両方で適切に表示される
