### 手動テスト

1. **未購入リストの並べ替え**
   - アイテムを上から下にドラッグ → 即座に新位置に表示されること
   - アイテムを下から上にドラッグ → 即座に新位置に表示されること
   - 元に戻る現象が発生しないこと

2. **購入済みリストの並べ替え**
   - 同上

3. **連続並べ替え**
   - 素早く複数回並べ替えを行っても正常に動作すること

4. **ネットワーク状態での確認**
   - オンライン時：並べ替え後Firestoreに正しく保存されること
   - オフライン時（ローカルモード）：並べ替えが正常に動作すること
