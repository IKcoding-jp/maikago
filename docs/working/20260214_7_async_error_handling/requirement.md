# 要件定義

**Issue**: #7
**作成日**: 2026-02-14
**ラベル**: bug, major

## ユーザーストーリー

## 要件一覧
### 必須要件
- [ ] `void`を返すasync関数を`Future<void>`に修正
- [ ] 重要な`catch (_)`にログ出力を追加
- [ ] エラーハンドリングの方針を統一

### オプション要件
- [ ] エラー通知の仕組み（DebugService経由の統一）

## 受け入れ基準
- [ ] `void` async関数が0件になっている
- [ ] 全`catch (_)`にログ出力がある
- [ ] `flutter analyze`でエラーがない
