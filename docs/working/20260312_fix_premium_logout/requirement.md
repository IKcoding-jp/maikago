# 要件定義

**Issue**: ログアウト後もプレミアム状態が維持されるバグ
**作成日**: 2026-03-12
**ラベル**: bug

## ユーザーストーリー
ユーザー「プレミアム購入済みアカウントでログアウトしたのに、ゲストモードでもプレミアム機能が使えてしまう」
アプリ「ログアウト時にプレミアム状態をリセットし、再ログイン時にFirestoreから復元する」

## 要件一覧
### 必須要件
- [ ] ログアウト時にOneTimePurchaseServiceのプレミアム状態をリセットする
- [ ] ゲストモードではフリー状態（プレミアム機能無効）になること
- [ ] 再ログイン時にFirestoreからプレミアム購入情報を読み込みプレミアム状態に復帰すること
- [ ] FeatureAccessControlがログアウト後に正しくフリー状態の制限を適用すること

### オプション要件
- [ ] ログアウト→再ログインのフローのユニットテスト

## 受け入れ基準
- [ ] ログアウト後、`isPremiumUnlocked` が `false` を返す
- [ ] ログアウト後、広告が表示される（`shouldShowAds()` が `true`）
- [ ] ログアウト後、テーマカスタマイズが制限される
- [ ] 再ログイン後、プレミアム状態が正しく復元される
