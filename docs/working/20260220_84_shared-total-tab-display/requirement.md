# 要件定義

**Issue**: #84
**作成日**: 2026-02-20
**ラベル**: design

## ユーザーストーリー
ユーザー「タブを共有すると『現在のタブ』と『共有合計』の2つの金額が出て分かりづらい。共有合計はタブのところに出して、下の金額は1つだけにしてほしい」
アプリ「共有合計金額をタブバーの共有グループタブに表示し、ボトムサマリーは『合計金額』のみ表示する」

## 要件一覧
### 必須要件
- [ ] タブバーの共有グループの端タブに共有合計金額を表示する
- [ ] ボトムサマリーの2段表示（「現在のタブ」+「共有合計」）を廃止する
- [ ] 共有モード・非共有モードともに「合計金額」のラベルで統一する
- [ ] ボトムサマリーの「合計金額」は現在のタブのアイテム合計のみを表示する

### オプション要件
- [ ] タブバーの共有合計表示のデザイン調整（フォントサイズ、配置等）

## 受け入れ基準
- [ ] 共有モード時、ボトムサマリーに金額が1つだけ（「合計金額」）表示される
- [ ] 共有モード時、タブバーの共有グループタブに共有合計金額が表示される
- [ ] 非共有モード時、従来通り「合計金額」のみ表示される
- [ ] 既存の予算機能・進捗バーに影響がない
