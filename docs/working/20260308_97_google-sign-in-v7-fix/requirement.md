# 要件定義

**Issue**: #97
**作成日**: 2026-03-08
**ラベル**: bug

## ユーザーストーリー
ユーザー「Googleアカウントでログインしたい」
アプリ「赤いSnackBarで『ログインエラーが発生しました』と表示される」→ ログインできない

## 要件一覧
### 必須要件
- [ ] Android版でGoogleログインが正常に動作すること
- [ ] Web版でGoogleログインが正常に動作すること
- [ ] ログイン後にFirestoreへのデータアクセスが正常に行えること
- [ ] ログアウト→再ログインが正常に動作すること

### オプション要件
- [ ] iOS版でのGoogleログインも確認（テスト環境がある場合）
- [ ] Windows版でのGoogleログインも確認

## 受け入れ基準
- [ ] Android: ログインボタンタップ → Google選択画面 → ログイン成功 → メイン画面遷移
- [ ] Web（デスクトップ）: ログインボタンタップ → ポップアップ認証 → ログイン成功
- [ ] Web（モバイル）: ログインボタンタップ → リダイレクト認証 → ログイン成功
- [ ] AccountScreen: ログアウト状態からのGoogleログイン連携が動作する
- [ ] flutter analyze でエラー・警告なし
- [ ] 既存テストがすべてパスする
