# タスク: 利用規約の改定

## 概要
現在のリポジトリの仕様に基づき、利用規約（`public/terms-of-service.html`）を改定する。
特に、以下の機能に関する条項を追加する。
1.  **有料サービス（プレミアムプラン/サブスクリプション）**
    *   リポジトリ内では「OneTimePurchase（買い切り）」として実装されているが、UIテキストに「7日間無料体験」「いつでも解約OK」等のサブスクリプション的な表現が含まれている。
    *   ユーザーの要望通り「サブスクリプションサービス」の条項を含めつつ、現在の買い切り仕様もカバーできる「有料サービス」として定義する。
2.  **家族共有機能（グループ共有）**
    *   `SharedGroupService` により提供される、複数ユーザー間での買い物リスト・予算の共有機能。
    *   共有に伴う責任、トラブル解決に関する免責事項を定義する。
