# 推奨コマンド

## 開発
- `flutter pub get` - 依存関係インストール
- `flutter analyze` - Lint / 静的分析
- `flutter test` - 全テスト実行
- `flutter test test/path/to_test.dart` - 単一テスト

## ビルド
- `flutter build apk --debug` - Android APK
- `flutter build web` - Web
- `flutter build windows` - Windows

## Git (Windows bash)
- `git status`, `git add`, `git commit -m "..."`, `git push`
- `gh issue view N`, `gh pr create`

## Firebase
- `cd functions && npm install` - Functions依存関係
- `firebase deploy --only functions` - Functionsデプロイ

## システム (Windows bash)
- `ls`, `cat`, `grep`, `find` - Unix互換コマンド（Git Bash経由）
